# 板卡说明

这个项目用于ARTIX-7 GTP USB3 WCH-MCU这块FPGA核心板，板上有一个QFN28封装的CH32V305,引脚兼容CH347 ,JTAG 引脚见下表：

| JTAG引脚 | 引脚编号 | GPIO编号 |
| -------- | -------- | -------- |
| TCK      |          | PB14     |
| TMS      |          | PB13     |
| TDI      |          | PB10     |
| TDO      |          | PD2      |

具体原理图见项目：xxxx

# 硬件兼容性

CH32V30x都是一个启动文件，除了Flash/RAM不一样可以直接用，这里也没用超，所以对于不同封装和不同型号，只需要修改对应引出的引脚即可。（注意WCH的手册，有的引脚内部是连接的）
